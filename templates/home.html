Django for Android is a simple framework for fast development of Web Apps using Django and (obviously) Python.
This project is based on Python for Android, thanks a lot to the team.

Install

pip install djangoforandroid

Configuration


INSTALLED_APPS = (
    ...
    'djangoforandroid.builder',
    'djangoforandroid.mdl', #optional, for mdl support
    'djangoforandroid.d4a', #optional, for urls support
)
